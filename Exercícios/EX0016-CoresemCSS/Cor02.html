Testando Gradiente

Nesta aula aprendemos a usar a tag (Background-image) que junto com a tag derivada (linear-gradiente) serve para usarmos efeitos css em degrades.
A seguir temos algumas tags importantes para usarmos com o linear gradient
To right = Para o efeito começar da direita
To left = Para o efeito começar da esquerda
To top = para começar de baixo
TO bottom = Para começar de cima
