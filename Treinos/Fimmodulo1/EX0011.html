Nesta aula aprendemos a utilizar imagens dinamicas, que se alteram de acordo com o tamanho do navegador
Tag Picture: serve para criar varias soucers, varias fontes de imagens, dentro dessas tags utilizamos a tag baixo
Tag souncer media: serve para definir vários tipos de arquivos em formato de mídia, max width para definir o mínimo para carregar uma nova foto, Exemplo max width: 1000px, que significa que acima de 1000px a foto da resoluçao original continuara source media="(min-width: )" srcset=""
