Para realizar um download basta utilizar a tag de link "a" e dentro dela a tag "download" e a tag type para informar o tipo do arquivo de download.

Conteudos da aula

Nesta aula aprendemos a utilizar links externos utilizando a tag "a" e aparecerá a tag href="" onde voce colocará o link desejado. dentro da tag "a" voce colocara as tags target e rel

Tag target: Utilizada com a tag self para abrir o link na própria aba. Com a tag blank para abrir em otra aba.

Tag rel: Utilizada com a tag external que indica que o link é externo, de outro site.
